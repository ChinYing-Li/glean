Build a JWE value from its elements and set to it.

Parameters

header - A variable-size JWE protected header.

key - A variable-size encrypted key. This can be an empty octet sequence.

initVector - A fixed-size, 96-bit, base64 encoded Jwe initialization vector. If not required by the encryption algorithm, can be an empty octet sequence.

cipherText - The variable-size base64 encoded cipher text.

authTag - A fixed-size, 132-bit, base64 encoded authentication tag. Can be an empty octet sequence.
