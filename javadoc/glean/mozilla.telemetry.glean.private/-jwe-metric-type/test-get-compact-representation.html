testGetCompactRepresentation

@JvmOverloads fun testGetCompactRepresentation(pingName: String = sendInPings.first()): String

Returns the stored value in the compact representation for testing purposes only. This function will attempt to await the last task (if any) writing to the the metric's storage engine before returning a value.

Parameters

pingName - represents the name of the ping to retrieve the metric for. Defaults to the first value in sendInPings.

Exceptions

NullPointerException - if no value is stored

Return
value of the stored metric
